```c
    max_fd = sockfd;
    return sockfd;
}
```

✅ Purpose:
Creates a **TCP socket** (`SOCK_STREAM`) using IPv4 (`AF_INET`) and adds it to the monitored file descriptor set.

---

### 🧩 5️⃣ Configure the server address

```c
struct sockaddr_in servaddr;
bzero(&servaddr, sizeof(servaddr));

servaddr.sin_family = AF_INET;
servaddr.sin_addr.s_addr = htonl(2130706433); // 127.0.0.1
servaddr.sin_port = htons(port);
```

* Fills in the address structure for the server.
* `2130706433` is `127.0.0.1` in integer form.
* `htons()` converts the port number from host byte order to network byte order (big-endian).
* This means your server will **listen on localhost**.

---

### 🧩 6️⃣ Bind the socket

```c
if (bind(sockfd, (const struct sockaddr *)&servaddr, sizeof(servaddr)))
    fatal_error();
```

* Binds the socket to the local IP and port you just defined.
* If another program is already using the port, `bind()` fails → `fatal_error()`.

---

### 🧩 7️⃣ Start listening for incoming connections

```c
if (listen(sockfd, SOMAXCONN))
    fatal_error();
```

* Tells the OS to start **listening** for client connections.
* `SOMAXCONN` = maximum number of queued pending connections.

---

### 🧩 8️⃣ The main server loop

```c
while (1)
{
    rfds = wfds = afds;
    if (select(max_fd + 1, &rfds, &wfds, NULL, NULL) < 0)
        fatal_error();
```

* This loop runs **forever**.
* `select()` waits until:

  * A socket is ready to read (`rfds`)
  * Or ready to write (`wfds`)
* When something happens (a new client connects or sends data), `select()` returns.

---

### 🧩 9️⃣ Handle ready file descriptors

```c
for (int fd = 0; fd <= max_fd; fd++)
{
    if (!FD_ISSET(fd, &rfds))
        continue;
```

* Loops through all open sockets.
* Checks which sockets are **ready for reading**.

---

### 🧩 10️⃣ Accept new clients

```c
if (fd == sockfd)
{
    socklen_t addr_len = sizeof(servaddr);
    int client_fd = accept(sockfd, (struct sockaddr *)&servaddr, &addr_len);
    if (client_fd < 0)
        fatal_error();
    register_client(client_fd);
    break;
}
```

* If the ready descriptor is the **main listening socket**, it means a client is trying to connect.
* `accept()` creates a **new socket** for that client.
* `register_client()`:

  * Assigns it an ID,
  * Adds it to `afds`,
  * Notifies other clients that a new one joined.

---

### 🧩 11️⃣ Receive and broadcast client messages

```c
int read_bytes = recv(fd, buf_read, 1000, 0);
if (read_bytes <= 0)
{
    remove_client(fd);
    break;
}
buf_read[read_bytes] = '\0';
msgs[fd] = str_join(msgs[fd], buf_read);
if (msgs[fd] == NULL)
    fatal_error();
send_msg(fd);
```

* Reads data from a connected client.
* If `recv()` returns 0 or −1 → client disconnected → `remove_client(fd)`.
* Otherwise, append data to the client’s message buffer (`msgs[fd]`).
* `send_msg(fd)` broadcasts the completed message to all other clients.

---

### 🧩 12️⃣ End of loop

After processing all sockets, the loop repeats indefinitely — the server keeps listening and forwarding messages forever.
